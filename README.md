# 🏡 AI House Price Prediction Platform

A premium full-stack AI-powered House Price Prediction platform built using React, FastAPI, XGBoost, Tailwind CSS, and Framer Motion.

This project combines modern machine learning with a cinematic SaaS-style UI inspired by Apple, Linear, Stripe, and OpenAI products.

---

## ✨ Features

* 🤖 AI-powered house price prediction using XGBoost
* ⚡ FastAPI backend for high-performance inference
* 🎨 Premium Apple-inspired UI/UX
* 🌌 Glassmorphism + animated gradient interface
* 📱 Fully responsive modern dashboard
* 🎭 Smooth Framer Motion animations
* 🧠 Intelligent property valuation system
* 🔥 Real-time prediction API integration
* 💎 Production-grade frontend architecture
* 🚀 Deploy-ready full-stack ML application

## 🚀 Run Locally

### Backend

```bash
conda activate nlp_course_env
python -m uvicorn main:app --reload
```

Backend runs on:

```text
http://127.0.0.1:8000
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
